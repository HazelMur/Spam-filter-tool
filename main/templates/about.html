About ByeHackers...

ByeHackers is a spam filter tool.

It provides protection and preventetion to all customers at the highest level possible.

ByeHackers aim is to help stop the fight against spam.

ByeHackers uses Machine Learning algorithms to classify an email as spam or ham.

The following Machine Learning algorithms are used:

Naive Bayes

NB algorithm is a supervised learning algorithm, which is based on Bayes theorem and used for solving classification problems

Support Vector Machine

SVM is a supervised machine learning algorithm which can be used for both classification or regression challenges. However, it is mostly used in classification problems

Random Forest

It is a flexible, easy to use machine learning algorithm that produces, a great result. It is also one of the most used algorithms, because of its simplicity and diversity as it can be used for both classification and regression tasks.

Logistic Regression

The algorithm is used for the classification problems. It is a predictive analysis algorithm and based on the concept of probability.
